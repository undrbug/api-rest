Documentación de la API REST

Rutas de Usuarios

Método Ruta Descripción
GET /api/users/count Obtiene el conteo total de usuarios.
GET /api/users Obtiene una lista de todos los usuarios.
GET /api/users/:id Obtiene un usuario específico por su ID.
GET /api/users/getbyemail/:email Obtiene un usuario específico por su email.

Rutas de Productos

Método Ruta Descripción
GET /api/products/drinklist Lista los tipos de bebidas disponibles.
GET /api/products Obtiene una lista de todos los productos.
GET /api/products/:id Obtiene un producto específico por su ID.
GET /api/products/search/:search Busca productos filtrados según el término de búsqueda.
